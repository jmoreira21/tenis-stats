# 🎾 ATP Stats

Site de estatísticas históricas do circuito ATP de tênis. Desenvolvido com Python (Flask) no backend e HTML/CSS/JavaScript puro no frontend.

---

## Funcionalidades

- **Buscar Jogador** — estatísticas de carreira, títulos por categoria, desempenho por superfície e histórico por temporada
- **Buscar Torneio** — histórico completo de campeões e vices de qualquer torneio
- **Campanha** — veja como um jogador se saiu em um torneio específico, incluindo qualificatória e tipo de entrada (Lucky Loser, Wildcard, etc.)
- **Head-to-Head** — confronto direto entre dois jogadores com comparativo de estatísticas de carreira
